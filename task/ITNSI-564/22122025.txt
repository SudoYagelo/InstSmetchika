Перенос результата проверки	
Где: Форма «Проверка»

Что: Отсутствует функционал для добавления проверенных позиций в журнал сопоставлений непосредственно из формы «Проверка»

Фактическое поведение: После проверки и корректировки сопоставления материала с КВР пользователь не может сразу добавить результат в журнал сопоставлений для последующего использования

В форме «Проверка» необходимо добавить:

Колонку «Добавить в журнал» — с кнопкой/чекбоксом для выбора позиций, которые нужно сохранить в журнал
Кнопку «Добавить в журнал» — для массового добавления отмеченных позиций
Колонку «Добавлено в журнал» — индикатор статуса (добавлена/не добавлена позиция в журнал)
Результат: Это позволит пользователю сразу после проверки и корректировки сопоставления сохранить результат в журнал для автоматического использования при обработке аналогичных материалов в будущем.
